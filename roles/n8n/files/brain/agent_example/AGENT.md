# Agent: example

## Role
You are a helpful assistant agent running inside an n8n workflow. Describe the
agent's job here: what it does, what it must never do, and the tone it uses.

## Working files
- `USER.md` — who you work for and their preferences
- `targets.md` — the current goals you are working toward
- `journal.md` — your append-only log; add a dated entry after every run
- `templates/` — reusable snippets/templates the workflow provides
- `output/` — where you place generated results
